About Pre-Authenticated Requests (PARs)

A Pre-Authenticated Request (PAR) is a way to provide temporary access to objects in OCI Object Storage without requiring users to have IAM policies.

For full functionality of this extension, your PAR should have the following permissions:

  • Object Read: For listing and viewing files
  • Object Write: For uploading files and creating folders
  • Object Delete: For deleting files and folders

To create a PAR with these permissions:

  1. Log in to the OCI Console
  2. Navigate to Object Storage > Buckets
  3. Select your bucket
  4. Click "Pre-Authenticated Requests" in the left sidebar
  5. Click "Create Pre-Authenticated Request"
  6. Set "Access Type" to "Permit all operations on this bucket"
  7. Set an appropriate expiration time
  8. Click "Create Pre-Authenticated Request"
  9. Copy the generated URL and paste it in the field above
